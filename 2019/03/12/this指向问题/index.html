this指向问题

Mar 12, 2019 | 技术
条评论
  • 普通函数的this指向在运行时确定
  • 箭头函数的this指向在定义之后就已经确定

一、test()

普通模式下this指向全局，严格模式下this为undefined

二、xx.test()

this绑定到函数调用者

三、test.call(xxx) / test.apply(xxx) / test.bind()

this指向arg

四、new test()

构造函数里的this指的就是将要被new出来的新对象

五、箭头函数 () => {}

this指向的是它的外层作用域this的指向
